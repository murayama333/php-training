# トレーニング

## a_array_tr1.php

次のプログラムがあります。

```php
<?php
$score = ["english" => 90, "math" => 88, "science" => 80];
```

次の実行結果となるようにプログラムを作成してください。

### 実行結果

```
$ php a_array_tr1.php
258
```

## a_array_tr2.php

次のプログラムがあります。

```php
<?php
$scores = [
  ["english" => 90, "math" => 88, "science" => 80],
  ["english" => 64, "math" => 72, "science" => 72],
  ["english" => 90, "math" => 92, "science" => 94]
];
```

次の実行結果となるようにプログラムを作成してください。

### 実行結果

```
$ php a_array_tr2.php
252
```

> `math`キーの値の合計を求めます。
